# Laravel(?) API Responses
This package is intend to help create standardized JSON responses using Laravel | PHP applications.

### Example Response Structure

```JSON
{
    "message": "User retrieved!",
    "data": {
        "Name": "Jhon Doe"
    },
    "status": {
        "code": 200,
        "message": "OK"
    },
    "meta": {
        "timestamp": "2024-09-21 14:36:24"
    }
}
```
